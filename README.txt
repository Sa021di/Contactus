"Функциональность"
1. Форма "Cвязаться с нами": Пользователи могут заполнять форму обратной связи, включая следующие элементы:
input с типами: email, text, checkbox, file, radio, reset
textarea
select
button

2. Проверка обязательных полей:
При отправке формы выполняется проверка обязательных полей. 
Если какие-либо обязательные поля остались незаполненными, пользователь получит соответствующее сообщение об ошибке.

3.Сохранение результатов в базе данных: 
Данные, отправленные через форму, сохраняются в базе данных для дальнейшего анализа и обработки.

4.Страница с результатами:
Пользователи могут просматривать список результатов обращений на странице "Результаты обращений". 
Список включает информацию о каждом обращении, включая дату отправки.

5.Отправка результатов на email администратора: 
Когда обращение отправляется, результаты также отправляются на указанный адрес электронной почты администратора.

позволяющей пользователям легко навигироваться между результатами, особенно если их большое количество.

Технологии
Open Panel Server
PHP_7.2
Apache_2.4-PHP_7.2-7.4
MySQL-8.0-Win.10
HTML: Фронтенд формы обратной связи и страницы результатов.
PHP: Обработка данных формы, взаимодействие с базой данных и отправка email.
MySQL: Хранение и управление данными обращений.
